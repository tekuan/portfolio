🏠 Inicio

Vamos criar e publicar um Formulário no GitHub Pages!

1. Criar uma Conta no GitHub (Windows, macOS, Linux)

  • Acesse o site GitHub.com.
  • Clique em Sign up no canto superior direito.
  • Insira seu e-mail, crie uma senha e escolha um nome de usuário.
  • Confirme seu e-mail e siga as instruções para finalizar a criação da conta.

2. Criar um Repositório no GitHub (Windows, macOS, Linux)

  • Após criar sua conta, clique no botão + no canto superior direito e selecione New repository.
  • Escolha um nome, como SEU_USUARIO.github.io, e marque a opção Public.
  • Marque Add a README file.
  • Clique em Create repository.

3. Instalar o Git (Windows, macOS, Linux)

  • Windows: Baixe o Git aqui e siga o processo de instalação padrão.
  • macOS: No terminal, execute o seguinte comando para instalar o Git:
    • xcode-select --install
  • Linux: No terminal, use o gerenciador de pacotes da sua distribuição:
  • Debian/Ubuntu:
    • sudo apt update
sudo apt install git
  • Fedora:
    • sudo dnf install git

4. Clonar o Repositório (Windows, macOS, Linux)

  • Abra o Git Bash (Windows) ou o Terminal (macOS/Linux).
  • Execute o comando abaixo, substituindo SEU_USUARIO pelo seu nome de usuário:
    • git clone https://github.com/SEU_USUARIO/SEU_USUARIO.github.io.git
  • Entre na pasta do repositório:
    • cd SEU_USUARIO.github.io

5. Criar um Formulário Simples em HTML (Windows, macOS, Linux)

Crie um arquivo index.html com o código HTML abaixo:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Formulário de Contato</title>
</head>
<body>
  <h1>Formulário de Contato</h1>
  <form action="https://formsubmit.co/seu-email@example.com" method="POST">
    <label for="name">Nome:</label>
    <input type="text" id="name" name="name" required><br>
    
    <label for="email">E-mail:</label>
    <input type="email" id="email" name="email" required><br>
    
    <label for="message">Mensagem:</label>
    <textarea id="message" name="message" required></textarea><br>
    
    <button type="submit">Enviar</button>
  </form>
</body>
</html>

Salve o arquivo index.html na pasta do repositório.

6. Fazer o Commit e Enviar para o GitHub (Windows, macOS, Linux)

  • No terminal, adicione o arquivo ao controle de versão:
    • git add index.html
  • Faça o commit do arquivo:
    • git commit -m "Adiciona formulário HTML"
  • Envie as alterações para o GitHub:
    • git push origin main

7. Ativar o GitHub Pages (Windows, macOS, Linux)

  • No GitHub, acesse o repositório https://github.com/SEU_USUARIO/SEU_USUARIO.github.io.
  • Vá em Settings (Configurações).
  • Na seção GitHub Pages, em "Source", selecione a opção main branch e clique em Save.
  • O GitHub gerará um link do seu site, que será algo como:
    • https://SEU_USUARIO.github.io/SEU_USUARIO.github.io/

8. Acessar o Formulário

Agora, seu formulário estará online no link gerado pelo GitHub Pages.
