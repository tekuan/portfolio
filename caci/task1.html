Como Criar uma Página HTML Básica

  1. Abra um Editor de Texto: Use um editor de texto como Bloco de Notas ou qualquer editor de sua escolha.
  2. Insira o Elemento HTML: O elemento raiz do documento HTML é <html>, que deve envolver todo o conteúdo. Exemplo:
    <html>
    </html>
  3. Adicione o Elemento <head>: O <head> contém informações sobre o documento, como estilos e scripts. Exemplo:
    <html>
      <head>
      </head>
    </html>
  4. Insira a Meta Tag de Codificação: Adicione a meta tag para definir a codificação de caracteres. Exemplo:
    <html>
      <head>
        <meta charset="utf-8">
      </head>
    </html>
  5. Adicione o Elemento <title>: O <title> define o título da página que aparece na barra do navegador. Exemplo:
    <html>
      <head>
        <meta charset="utf-8">
        <title>Minha primeira página web</title>
      </head>
    </html>
  6. Insira o Elemento <body>: O <body> contém o conteúdo visível da página. Exemplo:
    <html>
      <head>
        <meta charset="utf-8">
        <title>Minha primeira página web</title>
      </head>
      <body>
      </body>
    </html>
  7. Adicione o DOCTYPE: Declare a versão do HTML no início do documento. Exemplo:
    <!DOCTYPE html>
    <html>
      <head>
        <meta charset="utf-8">
        <title>Minha primeira página web</title>
      </head>
      <body>
      </body>
    </html>
  8. Salve o Documento: Salve o arquivo como index.html, com a codificação UTF-8.
  9. Visualize no Navegador: Abra o arquivo HTML no navegador para ver o resultado.
  10. Adicione Elementos <h1> e <p>: Insira um título e um parágrafo para exibir conteúdo na página. Exemplo:
    <!DOCTYPE html>

        <meta charset="utf-8">
